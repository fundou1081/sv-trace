# SV-TRACE 功能规划 V2
## 基于设计/验证/管理多视角的需求分析

---

## 视角概述

| 视角 | 关注点 |
|------|--------|
| **设计工程师** | RTL质量、可重用性、性能优化、功耗面积 |
| **验证工程师** | 覆盖率、bug发现效率、激励生成、回归 |
| **项目管理** | 质量指标、进度追踪、团队协作、报告 |

---

## 1. FSM状态机深度分析

### 设计视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **状态编码建议** | 根据状态数推荐 One-hot/Binary/Gray | P1 |
| **复杂度评分** | 自动识别过复杂状态机，提示拆分 | P1 |
| **死锁/不可达检测** | 提前发现设计bug | P1 |
| **跨时钟域同步建议** | 状态机跨CDC时的同步器推荐 | P2 |

### 验证视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **状态/跳转覆盖率** | 量化验证进度 | P1 |
| **验证计划自动生成** | 基于状态机生成验证点清单 | P2 |
| **SVA属性自动生成** | 生成assertion用于形式验证 | P2 |
| **随机约束建议** | 推荐状态机激励策略 | P3 |

### 管理视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **复杂度趋势追踪** | 各模块状态机复杂度历史对比 | P3 |
| **质量评分卡** | 设计质量一目了然 | P2 |
| **审查要点导出** | 评审会议资料自动生成 | P2 |

---

## 2. CDC多时钟域检测

### 设计视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **异步信号自动识别** | 快速定位跨时钟域信号 | P1 |
| **同步器推荐** | 根据场景推荐 2级FF/握手/mux同步方案 | P1 |
| **数据路径完整性** | 格雷码、延迟检测 | P1 |
| **时序约束生成** | 自动输出 set_clock_groups 等约束 | P2 |

### 验证视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **CDC覆盖率统计** | 哪些CDC路径已验证 | P1 |
| **常见模式检测** | 快到慢、慢到快、多bit同步场景 | P1 |
| **亚稳态风险评分** | MTBF估算 | P2 |
| **验证激励建议** | 推荐验证策略 | P2 |

### 管理视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **CDC问题汇总** | 按严重性分类的问题清单 | P1 |
| **风险评估报告** | 项目CDC风险量化 | P2 |
| **审查清单** | CDC审查要点 | P2 |

## 3. 复位完整性检查

### 设计视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **复位类型识别** | 同步/异步、极性、热复位 | P1 |
| **复位树分析** | 复位信号fanout、负载均衡 | P1 |
| **复位时序检查** | 释放顺序、亚稳态风险 | P2 |
| **上电序列建议** | 推荐的复位顺序 | P2 |

### 验证视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **复位覆盖率** | 验证是否覆盖所有复位场景 | P1 |
| **上电复位激励** | 自动生成上电测试用例 | P2 |
| **复位释放验证** | 验证复位释放时序 | P2 |

### 管理视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **复位域配置表** | 复位域统一管理视图 | P1 |
| **复位问题追踪** | 历史复位问题记录 | P2 |

## 4. 跨时钟域分析 (Timed Path)

### 设计视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **路径分类** | 同时钟/慢到快/快到慢 | P1 |
| **数据采样窗口** | setup/hold裕量分析 | P2 |
| **MTBF计算** | 亚稳态发生率估算 | P2 |
| **时序优化建议** | 流水线、寄存器插入建议 | P2 |

### 验证视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **时序收敛检查** | 所有路径是否满足时序 | P1 |
| **违例路径分析** | 关键违例定位 | P1 |
| **形式验证属性** | 生成时序SVA | P2 |

### 管理视角 🎯
| 功能 | 价值 | 优先级 |
|------|------|--------|
| **时序问题汇总** | 按模块/路径分类 | P1 |
| **设计评审要点** | 时序审查资料 | P2 |
