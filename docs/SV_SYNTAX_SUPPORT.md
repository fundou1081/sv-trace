| **过程块** | always_comb | ✅ 完全支持 | 组合逻辑 |
| **赋值** | <= (非阻塞) | ✅ 完全支持 | 时序逻辑 |
| **赋值** | = (阻塞) | ✅ 完全支持 | 组合逻辑 |
| **赋值** | assign (连续) | ✅ 完全支持 | 连续赋值 |
| **时钟** | posedge clk | ✅ 完全支持 | 上升沿 |
| **时钟** | negedge clk | ✅ 完全支持 | 下降沿 |
| **复位** | async (or negedge) | ✅ 完全支持 | 异步复位 |
| **复位** | sync (if rst) | ✅ 完全支持 | 同步复位 |
| **控制流** | if/else | ✅ 完全支持 | 条件赋值 |
| **控制流** | case | ✅ 完全支持 | 多路选择 |
| **生成** | generate for | ✅ 完全支持 | 循环生成 |

| 语法类别 | 语法元素 | 优先级 | 原因 |

| **控制流** | for loop | P1 | 循环内的赋值 |
| **控制流** | while loop | P2 | 较少使用 |
| **生成** | generate if | P1 | 条件生成 |

| **任务** | task | P1 | 任务内的赋值 |

| **接口** | interface | P2 | 接口赋值 |
| **断言** | assert | P3 | 验证断言 |

## 4. 改进计划

### P1 - 高优先级

1. **for loop 支持** - 循环内的赋值
2. **function 支持** - 函数内的赋值
3. **task 支持** - 任务内的赋值
4. **generate if 支持** - 条件生成

### P2 - 中优先级

1. **always @ 支持** - 旧语法兼容
2. **while loop 支持** - 循环赋值
3. **class 支持** - 面向对象
4. **interface 支持** - 接口赋值

### P3 - 低优先级

1. **assert 支持** - 验证断言
2. **do while 支持** - 循环读取
3. **foreach 支持** - 循环读取
