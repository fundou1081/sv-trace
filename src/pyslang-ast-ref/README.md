# pyslang AST 参考源代码

> 本目录包含未使用的语法解析器，仅供参考

## 概述

本目录收集了 sv-trace 项目中的 294 个未使用语法解析器。
它们作为 pyslang AST 结构的参考，不直接用于项目代码。

## 文件来源

原位于 `src/parse/` 目录，因未被 `__init__.py` 引用而移动。

## 使用原则

✅ **可以**:
- 查阅 AST 节点结构
- 理解 SyntaxKind 名称
- 学习语法解析方式

❌ **禁止**:
- 复制到项目代码
- 直接 import 用于生产

详见: [DEVELOPMENT_DISCIPLINE.md](../DEVELOPMENT_DISCIPLINE.md)
